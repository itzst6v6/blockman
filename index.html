Gytis's Mistake (How It Started)

Let's be clear: I didn't start this, but I'm going to finish it. The person who kicked this all off was my friend, Gytis. He made a dumb decision. For some apps—yes, really, just for some apps—he gave me the server engine. He was never supposed to give it to me, and that single act was a betrayal of our other friend, Vasu. That's how it all began.

My Reaction (How I Ended It)

Now, my part. I'm not innocent. I accepted the files, which made me part of the betrayal. My mistake was being naive and making a YouTube video about the 2020 engine. The chaos was instant. The fallout was fast. I was banned from Vasu's Discord project, 'BlockyVerse', for the video.

That was the final straw. The whole problem was that these engines were rare. That rarity is what people used for power. After the ban, I decided to destroy the one thing they cared about: the secrecy. I turned my server into a No Rules, Free Admin Land, and leaked the conversations back to Vasu. Now, I'm ending it for good by making these files public. If they aren't rare anymore, they can't be used as a weapon. The power trip is over.

Important Notice: The files offered here are precompiled. You are downloading ready-to-run game clients and servers, not the source code.

These precompiled versions work perfectly on Windows 11. However, to compile the engines from scratch, you would need the original source code (not included here) and would have to run Windows 10.
